## 失败教训

### Memory 目录混淆（2026-03-05）
**问题：** 系统中存在两个 memory 目录
- `~/.openclaw/memory/` → 系统级数据（SQLite、向量索引）
- `~/.openclaw/workspace/memory/` → 用户记忆文件

**错误：** 多次将文件写入系统目录

**解决机制：** 创建 PATH-CHECKLIST.md，写入前强制检查
**记忆口诀：** "工作区 = workspace，系统目录别乱写"
