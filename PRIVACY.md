| 文件位置 | 可见性 | 可否包含隐私 |

## 隐私信息清单（禁止公开）

### ❌ 不可出现在 GitHub 上

- [ ] 真实姓名（顾立宏 → 用 **LH** 代替）
- [ ] 公司全称（公司 → 用 **公司** 代替）
- [ ] 具体业务线（就业业务线 → 用 **业务线** 代替）
- [ ] 具体职位（算法 TL → 用 **技术岗位** 代替）
- [ ] 内部会议/研讨细节
- [ ] 任何可能识别身份的信息

### ✅ 可以公开

- [ ] 技术讨论
- [ ] 代码和项目
- [ ] 通用方法论
- [ ] 学习记录
- [ ] 错误反思（不涉及具体业务）
