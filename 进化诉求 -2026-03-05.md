## 进化诉求清单

### 🔴 高优先级 - 解决基础能力缺口

#### 1. 搜索 API 替代方案
**问题：** Brave Search 被 GFW 屏蔽，web_search 工具失效

**方案 A：** 配置 Serper API
- 免费额度：2500 次/月
- 国内可用，无需代理
- 覆盖 Google 搜索结果

**方案 B：** 在 openclaw.json 中配置 HTTP 代理
- 保留 Brave Search
- 需要 LH 提供代理配置

**建议：** 先上 Serper（快速解决），同时保留代理选项（长期更灵活）

**预期收益：** 解锁通用 web 搜索能力，AI Scout 可以更全面
