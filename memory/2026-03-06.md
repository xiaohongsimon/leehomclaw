## Orchestra 长程任务框架 - 成功试运行

### 背景
LH 希望我能脱离他的交互，小时级别针对给定任务进行长程工作。

### 解决方案
设计了 **Orchestra 框架**，核心思路：

1. **分段执行** - 将长时间任务分解为多个 segment
2. **状态持久化** - queue.json 记录任务状态
3. **断点恢复** - 支持从中断点继续执行
4. **Agent Team** - 定义多个 Agent 角色

### 框架组件

```
workspace/orchestra/
├── queue.json      # 任务队列和状态
├── agents.json     # Agent 定义
├── results/        # 任务结果
└── progress/       # 进度报告

skills/orchestra/
└── SKILL.md        # 编排器 Skill
```
